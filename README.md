# Business Intelligence SaaS Platform

## Quick Deploy

1. **Upload to GitHub**
2. **Connect to Vercel**
3. **Add environment variables:**
   ```
   DATABASE_URL=your-supabase-connection-string
   JWT_SECRET=your-random-secret-32-chars-long
   GOOGLE_PLACES_API_KEY=your-google-api-key
   ```
4. **Deploy**
